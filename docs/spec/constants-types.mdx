## API Limits (`constants/apiLimits.ts`)

### Image Limits

| Constant | Value | Description |
|----------|-------|-------------|
| `API_IMAGE_MAX_BASE64_SIZE` | 5 MB | Max base64-encoded image size |
| `IMAGE_TARGET_RAW_SIZE` | 3.75 MB | Target raw image size |
| `IMAGE_MAX_WIDTH` | 2000 px | Client-side max width |
| `IMAGE_MAX_HEIGHT` | 2000 px | Client-side max height |

### PDF Limits

| Constant | Value | Description |
|----------|-------|-------------|
| `PDF_TARGET_RAW_SIZE` | 20 MB | Max raw PDF size |

| `PDF_MAX_EXTRACT_SIZE` | 100 MB | Max PDF for extraction |
