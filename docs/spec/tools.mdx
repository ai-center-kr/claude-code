## Tool Framework

### Core Interface (`Tool.ts`)

```typescript
type Tool<Input, Output, Progress> = {
  // Identity
  name: string
  isMcp?: boolean
  alwaysLoad?: boolean
  shouldDefer?: boolean

  // Schema
  readonly inputSchema: Input
  readonly outputSchema?: ZodType<Output>

  // Metadata
  description(): Promise<string>
  prompt(): Promise<string>
  userFacingName(input?): string

  // Capability flags
  isEnabled(permissionContext?): boolean
  isConcurrencySafe(input?): boolean
  isReadOnly(input?): boolean
  isDestructive?(input): boolean

  // Execution
  validateInput?(input): Promise<ValidationResult>
  checkPermissions(input, context): Promise<PermissionDecision>
  call(input, context): Promise<ToolResult<Output>>

  // UI rendering
  renderToolUseMessage(input, options): ReactNode | null
  renderToolResultMessage(output, ...): ReactNode | null
}
```

### buildTool() Defaults

| Field | Default |
|-------|---------|
| `isEnabled` | `() => true` |
| `isConcurrencySafe` | `() => false` |
| `isReadOnly` | `() => false` |
| `checkPermissions` | `async () => ({ behavior: 'allow' })` |

### ToolUseContext

The context passed to every tool call:

```typescript
type ToolUseContext = {
  options: { commands, tools, mcpClients, mainLoopModel, thinkingConfig }
  abortController: AbortController
  getAppState(): AppState
  setAppState(fn): void
  readFileState: Map<string, { mtime: number; content: string }>
  permissionContext: ToolPermissionContext
  setToolJSX: SetToolJSXFn
  onPermissionRequest(request): Promise<PermissionDecision>
  agentId?: AgentId
  isSubagent?: boolean
}
```

## Permission System

```typescript
type ToolPermissionContext = {
  mode: PermissionMode
  // 'default' | 'plan' | 'auto' | 'bypassPermissions' | 'acceptEdits'
}
```

Each tool's `checkPermissions()` returns a `PermissionDecision`:
- `allow` - proceed with execution
- `deny` - block with reason
- `ask` - prompt user for approval
